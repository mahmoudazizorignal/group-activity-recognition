# Checkpoints

Large model checkpoints are hosted as GitHub Release assets to avoid exceeding Git LFS quotas.

To download the B8 checkpoint used in this repository, visit the release:

https://github.com/mahmoudazizorignal/group-activity-recognition/releases/tag/checkpoints-b8-20251223

After downloading, place the file in `checkpoints/b8/` with its original name or the name used in the release.
